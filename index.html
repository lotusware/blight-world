► Changelog

    January 25, 2020
  • [QUALITY OF LIFE] Added a Knack indicator
    Line of text to add clarity and to better describe the mechanic. When active, a bonus is added to each check to demonstrate what the check will be with the added Knack. Removed impossible and difficult values from the Knack trait as these trickle down in the other stats anyway.
  • [BUGFIX] Items can now stack above 99
    When stacked more than 99 it would break the cascading array to determine if the value was a quantity or an item. Fixed the array so it uses odd/even columns. Items at 99 stacks have been rounded up to 100.
  • [NEW] Added Shots Left indicator in Combat menu
    Function to cross reference ammo requirements on equipped weapon with what is held in player's inventory.
  • [QUALITY OF LIFE] Moved Accuracy to combat screen and removed IMPOSSIBLE checks
    Impossible accuracy checks are grouped with Difficult accuracy checks. Because its referenced more in combat, moved the trait to the Combat menu.
  • [MISC] Text changes
    Changed wording for added clarity on how attributes affect rolls, changed "Photon Canisters" to just "Photon" in order to work with the ammo mechanic

      • January 23, 2020
    • [REMOVED] Ranged, Melee, Flux Bonus roll mechanic. Weapons adjusted to compensate.
      Items with bonus rolls often had scaling baked into the item. Rather than have two stats for each attribute to keep track of, item scaling is unified under each respective stat.
    • [BUGFIX] Datalog links (green) are clickable in landscape mode
      The static navigation and status area was creating an invisible overlay that appeared over top of the logs when held in landscape mode. Changed how these are displayed to remove the invisible wall.
